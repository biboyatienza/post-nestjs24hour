# The Aftermath: 24 Hour NestJS API Challenge 

## Description
The goal of this test is to create a consumable RESTful API for storing and retrieving
images.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
